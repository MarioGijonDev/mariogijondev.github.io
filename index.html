Proyectos destacados

Estos son los main projects de mi portfolio de github y se encuentran en constante desarrollo

IA En la web

Combino la web con la inteligencia artificial creando una aplicación capaz de, con solo mostrar los gestos a la WebCam, modificar el volumen del dispositivo, detectar la posición de los dedos para saber que gesto se realiza... etc. Usando librerías de Python como MediaPipe, Pandas (para el procesamiento de datos) y Flask, entre otros, junto con Socket.io, para crear una conexión concurrente entre cliente y servidor. Es un proyecto para el TFGS, el diseño aún está por terminar.

Prueba técnica React

Aprendí a realizar dos fetch de datos con ajax, dependientes uno del otro a dos apis diferentes, junto con hooks de React como useState, useEffect, custom hook y test end to end para un código escalable.

Taller WebDesign Bootstrap 5

Un taller online donde realicé el diseño de una aplicación web responsiva con bootstrap, simulando la posible página web de un cliente.

Tic Tac Toe

Puse en práctica la abstracción de componentes y lógica de la aplicación, diseño responsivo además del uso del LocalStorage para almacenar datos de la sesión del juego.

ApiRest php

Generé y consumí una api con los principios Rest, utilizando JWT para la autentificación, PDO para la conexión a la base de datos MySQL y .htaccess para mayor legibilidad de la url con el módulo REWRITE.
